HTML <br>

<br>태그는 텍스트 내의 줄바꿈(line-break)을 정의할 때 사용합니다.

HTML소스코드 내에서는 엔터(Enter)를 사용하여 줄바꿈을 실시하여도 브라우저 화면 내에 실제로 적용되지는 않습니다.
따라서 <br> 요소는 주소(address)나 시(poem)등 줄바꿈이 중요한 텍스트에 유용하게 사용됩니다.

<br>요소는 빈 태그(empth)로, 종료 태그가 존재하지 않습니다.

예제 
                <p>
                    HTML은 HyperText Markup Language의 약자입니다.
                    웹 페이지는 HTML 문서라고도 불리며,
                    HTML 태그들로 구성됩니다.
                </p>
                
                <p>
                    HTML은 HyperText Markup Language의 약자입니다.<br>
                    웹 페이지는 HTML 문서라고도 불리며,<br>
                    HTML 태그들로 구성됩니다.
                </p>
