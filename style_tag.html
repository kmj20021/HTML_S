HTML <style>

<style>태그는 해당 HTML문서의 스타일 정보를 정의할 때 사용합니다.

<style>요소는 해당 요소가 포함된 HTML문서의 콘텐츠에 적용되는 CSS를 명시하고 있으며, HTML문서는 여러 개의 <style>요소를 포함할 수 있습니다.
별도의 파일로 저장된 외부 스타일 시트는<link>요소를 사용하여 참조할 수 있습니다.

예제 
                <head>
                    <style>
                        body { background-color: lightpink }
                        h1 { color: red; }
                        p { text-decoration: underline; }
                    </style>
                </head>
