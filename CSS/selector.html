선택자(Selector)

선택자(Selector)란?
선택자란 말 그대로 선택을 해주는 요소입니다. 이를 통해 특정 요소들을 선택하여 스타일을 적용할 수 있게 됩니다.

선언(Declaration) = 속성(Property): 속성값(Property Value)

  • 항상 마지막에 세미콜론(;)을 작성해 주어야 한다.

예제

스타일 테그
<head>
   선택자 {
           속성(Property) : 속성값(Property Value)
   }
</head>
스타일 속성
